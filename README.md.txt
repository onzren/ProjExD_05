# こうかとん探検隊
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
主人公キャラクター(こうかとん)を十字キーを使って操作し、敵をすべて倒したらクリア

## ゲームの実装
###共通基本機能
* 主人公キャラクターに関するクラス
* 撃つ球に関するクラス
* ブロックに関するクラス
* 敵に関するクラス
* マップに関するクラス
### 担当追加機能
* トラップの作成
* 敵の射撃追加
